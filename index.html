Kuis — —

Kuis dijalankan melalui Google Apps Script yang terhubung ke Google Spreadsheet. Hubungkan URL Apps Script Web App kamu di sini.

ℹ️ Cara menghubungkan kuis

1. Buat Google Spreadsheet berisi soal kuis.
2. Buka Extensions → Apps Script, buat code.gs dan index.html.
3. Deploy sebagai Web App (akses: Anyone).
4. Salin URL Web App, tempel ke variabel KUIS_URL di index.html ini.
5. Kuis akan tampil di dalam halaman ini secara otomatis.
